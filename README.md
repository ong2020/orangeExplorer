# The Orange-Block-Explorer
The Orange Block Expolorer

## How to start?
```bash
$ npm install
$ ORANGE_RPC_URL=<YOUR_RPC_URL> npm start
```

## What features are there?
This app use [`web3.js`](https://github.com/ong2020/web3.js/) library.

* Block info search 
* Transaction info search 
* Address info search
